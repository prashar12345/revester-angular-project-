On computer networks, the upload is to send data to a remote system, such as a server or other client, so that the remote system can store a copy.

Remote Upload

Transferring data from one remote system to another under the control of a local system is remote uploading.

Remote uploading is used by some online file hosting services. It is also used when the local computer has a slow connection with remote systems, but there is a fast connection between them. Without remote uploading functionality, you will first need to download data to a local host, and then upload it to a remote host server, both times through slow connections.
